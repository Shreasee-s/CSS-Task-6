# Login Page UI

A modern and responsive login page built using HTML and CSS.

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Open `index.html` in your browser or run it using VS Code Live Server.

## Live Demo

https://shreasee-s.github.io/CSS-Task-6/

## Preview

This project features:

- Split-screen layout
- Product illustration panel
- Gradient background
- Email and password input fields
- Forgot Password link
- Login button with hover effects
- Register Now call-to-action
- Flexbox and CSS Grid based layout

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid

## Features

### Layout
- Two-panel design
- Left panel displays a product image
- Right panel contains the login form

### Form Components
- Email input field
- Password input field
- Forgot Password link
- Login button
- Register Now link

### Styling
- Linear gradient background
- Custom typography
- Responsive spacing using CSS Grid
- Hover effects on buttons and links
- Rounded buttons and inputs

## Concepts Practiced

This project helped reinforce:

- Semantic HTML structure
- Flexbox alignment
- CSS Grid layouts
- Form creation and styling
- Hover states and pseudo-classes
- Custom fonts
- Responsive design fundamentals
